# Juego de Preguntas

Este proyecto es un juego interactivo de preguntas que desafía a los jugadores a responder cinco preguntas al azar dentro de un límite de tiempo de un minuto. El juego incluye un temporizador, botones para finalizar o reiniciar el juego, y un diseño profesional.

## Estructura del Proyecto

El proyecto contiene los siguientes archivos:

- `src/index.html`: Estructura HTML del juego.
- `src/style.css`: Estilos CSS para un diseño atractivo.
- `src/main.js`: Lógica del juego, incluyendo la selección de preguntas y el temporizador.
- `README.md`: Documentación del proyecto.

## Instrucciones para Ejecutar el Juego

1. Clona este repositorio en tu máquina local.
2. Navega a la carpeta `juego-preguntas/src`.
3. Abre el archivo `index.html` en tu navegador web.
4. Disfruta del juego y trata de responder todas las preguntas antes de que se acabe el tiempo.

## Características

- **Preguntas Aleatorias**: Cada vez que inicias el juego, se seleccionan cinco preguntas al azar.
- **Temporizador**: Un temporizador de un minuto que cuenta regresivamente mientras juegas.
- **Botones de Control**: Dos botones que te permiten finalizar el juego o volver a intentarlo.
- **Diseño Profesional**: Estilos CSS que mejoran la experiencia visual del juego.
